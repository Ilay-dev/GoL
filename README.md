# Game of Life

![Screencast_20260206_233744](https://github.com/user-attachments/assets/289f16fb-f5cb-4447-9e47-07c29eca9bbc)


A high-performance, infinite-grid implementation of Conway's Game of Life. Built with vanilla JavaScript and HTML5 Canvas, featuring a dark architectural aesthetic and liquid glass UI.

## Features

- **Infinite Canvas:** Uses a sparse matrix to allow indefinite pattern growth.
- **Design:** Deep charcoal palette with dynamic vignette and frosted glass interface.
- **Controls:** Variable simulation speed (x1 to x1000) and zoom/pan support.

## Controls

- **Left Click:** Draw / Erase cells
- **Middle Click + Drag:** Pan view
- **Scroll Wheel:** Zoom in / out
- **UI:** Control speed, play/pause, and clear grid
